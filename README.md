# Producer-Schedule
アイドルマスターの公式イベントスケジュールをGoogleカレンダーに登録するための、Googleスプレッドシート用Google App Script。

# Feature
- 以下公式サイトをスクレイピングしイベントの予定をGoogleスプレッドシートに登録する
    - https://idolmaster-official.jp/schedule
- 1枚のシートで年間のイベントを管理する
- 実行した日から3か月先までのスケジュールを取り込む
- イベント情報の更新やキャンセルを反映する
- イベントの情報をGoogleスプレッドシートからGoogleカレンダーに登録する
- イベントに関連するアイドルマスターのブランド毎にカレンダーを管理する
    - 対象のブランドは以下
        - THE IDOL M@STER
        - シンデレラガールズ
        - ミリオンライブ！
        - SideM
        - シャイニーカラーズ
        - 学園アイドルマスター
        - その他
    - カレンダー名を「P予定表（ブランド名）」とし、存在しない場合は新規作成する
    - 一つのイベントが複数のブランドに関連する場合、それぞれのカレンダーに予定を登録する
- イベントに開始時間が設定されている場合、1時間の予定として登録する
- イベントに開始時間が設定されていない場合、終日の予定として登録する
